# Embedded Linux Kernel Lab

Hands-on Embedded Linux and Linux Kernel development on Raspberry Pi 4 using Yocto Project. Covers custom image creation, kernel modules, memory management, procfs, device drivers, and Edge AI integration.

## Topics

- Yocto Project
- Custom Linux Image
- Raspberry Pi BSP
- Linux Kernel Modules
- Procfs
- Memory Management
- Device Drivers
- Embedded Linux Debugging
- Edge AI
